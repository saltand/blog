1. $route 和 $router 的区别 #

$route 是路由信息对象，包括 path，hash，query，fullPath，matched，name 等路由信息参数； $router 是路由实例对象，包括了路由的跳转方法，实例对象等

3. 什么是 vue 生命周期？ #

阶段方法名方法名
初始化beforeCreatecreated
挂载beforeMountmounted
更新beforeUpdateupdated
销毁beforeDestroydestroyed
  1. beforeCreate
    实例创建完毕，data 和 methods 尚未被初始化
  2. created
    最早能够访问 Vue 实例中保存的数据和方法的地方
  3. beforeMount
    表示 Vue 已经编译好了最终模板，但是还没有将最终模板渲染到界面上。
  4. mounted
    Vue 已经完成了模板的渲染，表示我们已经可以拿到界面上渲染之后的内容了。
  5. beforeUpdate
    表示 Vue 实例中保存的数据被修改了。此时界面还未修改
  6. updated
    数据和界面都同步更新之后就会调用 updated

    beforeUpdate 和 updated 函数在组建创建过程中不会自动触发，只有在 DOM 发生改变时才会触发

  7. beforeDestroy
    表示当前组件即将被销毁,beforeDestroy 函数是我们最后能够访问到组件数据和方法的函数。

    只要组件不被销毁，那么 beforeDestroy 就不会调用

    beforeDestroy 阶段可以解绑一些全局事件，以及移除当前组件定时器

  8. destroyed
    表示当前组件已经被销毁
